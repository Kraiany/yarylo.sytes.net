How a Mailing List Works
========================

Basically, it goes something like this...

   1.Members send an Email to a special address. 
   2.The Mailing List server receives the Email. 
   3.The Mail Server sends a copy of that message to each registered 
member.

The "special address" is decided by the list's administrator. 
example@ml.asahi-net.or.jp is an example of how the mailing list 
address will look.
The following restrictions apply: 

   1.The part to the right of "@" can only be ml.asahi-net.or.jp. 
   2.The part to the left of "@" is decided by the List Administrator. 
It can only contain English alphabet letters, numbers and hyphens. It 
can not begin with a
     hyphen. 

Any Email messages sent to asahi-club@ml.asahi-net.or.jp will be 
forwarded to all members of that mailing list.


Mailing List - User's Manual
============================

This document describes how Users can use the mailing list.

About the Mailing List Server

The Mailing List server software used at Asahi Net is based on software 
develped by Great Circle & Associates [ Majordomo ] version 1.93.

various changes to the version used at Asahi Net.

Warning:

     This manual describes how to use the Mailing List service offered 
by Asahi Net. Procedures explained within may or may not work on Mailing
     Lists offered by other service providers. 



Sending Mail to the Mailing List

To send mail to the Mailing List, simply send an Email to that mailing 
List's address. For example: To send a message to a mailng list called
hypothetical@ml.asahi-net.or.jp, use the BBS or your favorite email 
software, type hypothetical@ml.asahi-net.or.jp in the To:
field, enter a subject and a message, and send. Your message will be 
delivered to everyone on the mailing list.

If the list is configured to so that the administrator has to approve 
each message, it will first be forwarded to the administrator, and if 
approved, posted to the list.

Command Mail

There are several other configurable services of the mailing list. Most 
of these can be set by sending a "command mail" to a designated email 
address. For example,
signing up to a mailing list requires the use of a "command mail". 
Command Mails contain simple commands of one word or more on the first 
line of the email
message. The Subject: is irrelevant. 

Command mails should be sent to the following address: 

     Majordomo@ml.asahi-net.or.jp 

Do not send command mails to the mailing list address.

Please see the Command List for a list of all available commands.
